手机端操作步骤

通用前提（所有手机必看）

1. 必须使用带数据传输功能的原装数据线，仅充电线无法识别usb0网卡；
2. 路由器有USB3.0接口优先插入，可获得更高网速；若2.4G WiFi出现干扰，可换USB2.0接口；
3. 建议优先插入路由器主板原生USB口，避免前置扩展口供电不足导致断连；
4. 插拔设备后请刷新页面更新网卡列表；操作过程保持手机亮屏，防止休眠断开。

安卓手机操作步骤

1. 用数据线连接手机与路由器USB口；
2. 下拉手机通知栏，点击「USB已连接」通知，进入后开启 USB网络共享/USB绑定；
3. 若无通知弹窗：打开手机设置 → 连接与共享/移动网络 → 个人热点 → 开启「USB网络共享」。

iPhone（iOS）操作步骤

1. 使用苹果原装Lightning/USB‑C数据线连接路由器，第三方劣质线大概率无法识别；
2. 手机弹出「是否信任此电脑」弹窗，点击信任，输入锁屏密码确认；
3. 打开手机设置 → 个人热点，开启热点开关，连接方式选择仅USB；
4. 等待3~10秒，路由器即可自动识别USB网卡。

常见坑：未点信任、同时开启WiFi热点、线材仅供电，都会导致识别失败。

插件后台后续操作

1. 本页面选择对应手机类型，下拉选中识别到的USB网卡；
2. 按需勾选IPv6、掉线自动重连，设置检测参数；
3. 点击「一键应用配置」，下方状态面板会实时刷新连接结果；
4. 如需恢复原有宽带上网，点击「删除全部USB共享配置」即可一键复原。
